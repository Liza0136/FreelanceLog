# FreelanceLog

Консольное приложение для учёта фриланс-проектов, клиентов, доходов и аналитики.

## Функциональность

- ✅ Управление проектами (создание, просмотр, редактирование, удаление)
- ✅ Фильтрация по клиенту, категории, статусу
- ✅ Ежемесячная финансовая отчётность
- ✅ Текстовая визуализация дохода с диаграммой (█)
- ✅ Статистика по категориям
- ✅ Прогноз дохода на текущий месяц
- ✅ Экспорт в CSV
- ✅ Экспорт в ZIP (CSV + JSON + резервная копия БД)
- ✅ Импорт из ZIP с восстановлением
- ✅ Автоматическое резервное копирование
- ✅ Логирование всех действий
